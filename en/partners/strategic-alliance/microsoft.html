Strategic Alliance: Red Hat and Microsoft

What happens when Red Hat and Microsoft join forces? You win.

The wait is over

Use Red Hat products on Microsoft Azure

Security, consistency, reliability: These are why enterprises trust Red Hat® solutions. They're also why enterprises turn to a public cloud like Microsoft Azure. It's only natural that customers want to unite the 2.

Now you can. Red Hat and Microsoft are teaming up to offer open hybrid cloud solutions on Azure.

Read the press release

This partnership is a powerful win for enterprises, ISVs and developers. With this partnership, we are expanding our commitment to offering unmatched choice and flexibility in the cloud today, meeting customers where they are so they can do more with their hybrid cloud deployments – all while fulfilling the rigorous security and scalability requirements that enterprises demand.

Scott Guthrie
Executive Vice President, Microsoft's Cloud + Enterprise division

Benefits

It's all about choice and simplicity

  • Easy-to-move subscriptions

    Red Hat Cloud Access lets you move your Red Hat subscriptions to Azure in 3 easy steps.

  • Choice without complexity

    Choose where to run your applications—in your datacenter and in the cloud—expanding your IT options without adding complexity.

  • Comprehensive support

    Get global support across hybrid environments from Red Hat and Microsoft in 18 regions.

  • New application development capabilities

    Enterprise and independent software vendor (ISV) developers can target their apps developed on Microsoft .NET to infrastructure powered by Red Hat Enterprise Linux®, including OpenShift by Red Hat.

  • Manageability

    We're working to expand Red Hat CloudForms' large set of existing infrastructure and cloud providers to integrate Azure and further expand Microsoft System Center 2012 Virtual Machine Manager. This will let you manage workloads across physical, virtual, and private and public clouds.

  • Portability

    Red Hat Enterprise Linux on Microsoft Azure provides the consistent enterprise platform and application programming interfaces (APIs) for certified apps and containers, resulting in portability across physical, virtual, and private and public clouds.

What is

Cloud terms and technologies

What is open hybrid cloud?

Open hybrid cloud is an IT approach that uses standardized, interoperable, modular technology. You can create a unified environment that lets you take advantage of existing resources and be open to new advances.

Learn more

What is Microsoft Azure?

Azure is a cloud platform from Microsoft. It includes compute, storage, data, networking, and application development tools in a combination of managed and unmanaged services that let you build, deploy, and manage apps any way you like.

Frequently asked questions

You have questions. We have answers.

What's the scope of the announcement between Red Hat and Microsoft?

Microsoft and Red Hat are working together to:

  • Make Red Hat products globally available on Azure.
  • Bring your existing Red Hat software subscriptions to Azure, letting you maintain your existing Red Hat support relationship, consistent service, and predictable pricing.
  • Integrate Red Hat CloudForms with Azure and Microsoft System Center.
  • Provide secure, consistent support no matter where the issue originates.
  • Distribute and support .NET within both Red Hat Enterprise Linux and OpenShift by Red Hat, including enabling containers for both platforms.
  • Extend the cross-certification of Red Hat products on Microsoft Hyper-V and Microsoft offerings on Red Hat Enterprise Virtualization.

When will the offerings be available?

Red Hat Cloud Access is available to Red Hat customers now, enabling them to easily import Red Hat products and use them in Azure. Preconfigured pay-as-you-go Red Hat offerings in the Azure Marketplace will be available in the months to follow.

What are the top customer use cases and scenarios for the offerings included in this announcement?

We expect the top use case to be Red Hat customers who want to use their subscriptions on Azure. They'll likely move initial Red Hat Enterprise Linux-based workloads to run on Azure, including websites, production workloads, and test and development requirements. Customers will prioritize cloudbursting scenarios, such as retailers that run Red Hat Enterprise Linux addressing growth and seasonality needs by taking advantage of Azure to run their apps when capacity demand spikes. We also expect customers to use Red Hat Enterprise Linux on Azure to prove out hybrid scenarios and new technologies, like containers, via a proof of concept.
