Beim Red Hat Konto anmelden

Anmelden

Über Ihr Red Hat Konto können Sie auf Ihr Benutzerprofil, Ihre Einstellungen und die folgenden Services abhängig von Ihrem Kundenstatus zugreifen:

  • Customer Portal
  • Red Hat Connect für Geschäftspartner
  • Benutzerverwaltung
  • Certification Central
Registrieren

Noch nicht registriert? Folgende Gründe sprechen dafür, dass Sie es sein sollten:

  • Greifen Sie auf Artikel in unserer Knowledgebase zu, verwalten Sie Ihre Supportfälle und Subskriptionen, laden Sie Updates herunter, und nutzen Sie viele weitere Funktionen über eine zentrale Schnittstelle.
  • Lassen Sie sich die Benutzer aus Ihrem Unternehmen anzeigen, und bearbeiten Sie deren Kontoinformationen, Einstellungen und Berechtigungen.
  • Verwalten Sie Ihre Red Hat Zertifizierungen, sehen Sie Ihre Prüfungsübersicht ein, und laden Sie Logos und Dokumente zum Thema Zertifizierung herunter.
Profil und Einstellungen bearbeiten

Über Ihr Red Hat Konto können Sie auf Ihr Benutzerprofil, Ihre Einstellungen und andere Services abhängig von Ihrem Kundenstatus zugreifen.

Vergessen Sie zu Ihrer Sicherheit nicht, sich wieder abzumelden, wenn Sie die Red Hat Services auf einem öffentlichen Computer verwendet haben.

Plataformas Linux

Red Hat Enterprise Linux
Es la base para las arquitecturas de nueva generación, gracias a la compatibilidad con todas las plataformas de hardware principales y miles de aplicaciones comerciales y personalizadas.
Red Hat Satellite
Es la forma más fácil de gestionar Red Hat® Enterprise Linux®: permite que sus sistemas sigan funcionando correctamente, estén debidamente protegidos y en consonancia con diversos estándares.
Red Hat Enterprise Linux OpenStack Platform
Plataforma segura y escalable para la creación de nubes privadas y públicas.

Relacionado

Programa Red Hat Enterprise Linux Developer
El programa para desarrolladores de Red Hat Enterprise Linux ofrece herramientas de desarrollado líderes del sector, recursos formativos y un ecosistema de expertos que ayudarán a los programadores de Linux a crear extraordinarias aplicaciones de Linux.
Preguntas más frecuentes sobre Red Hat Satellite

Red Hat JBoss Middleware

Acelerar

Red Hat JBoss Enterprise Application Platform
Contenedor de Java™ EE 6 totalmente certificado que incluye todo lo necesario para crear, ejecutar y gestionar los servicios basados en Java.
Red Hat JBoss Web Server
Solución única tanto para páginas web a gran escala como para aplicaciones web Java ligeras. Incluye las versiones preparadas para producción y certificadas de Apache Web Server, Apache Tomcat y los conectores habituales que se usan entre ambos.
Red Hat JBoss Data Grid
Solución de almacenamiento en caché inteligente y distribuido que amplía aplicaciones de forma flexible mediante el acceso fiable y rápido a datos de uso frecuente.
Red Hat JBoss Portal
Marco para el desarrollo de páginas web colaborativas y con un gran nivel de integración, así como aplicaciones de autoservicio que crean experiencias de usuario atractivas.
Red Hat JBoss Developer Studio
Entorno de desarrollo integrado (IDE) certificado y basado en Eclipse para desarrollar, probar e implementar aplicaciones web avanzadas, aplicaciones web móviles, aplicaciones empresariales transaccionales y aplicaciones y servicios de integración basados en la arquitectura orientada a servicios (SOA).
Red Hat JBoss Operations Network
Componente clave de nuestras soluciones gestionadas de middleware que proporciona un punto único de control para implementar, gestionar y supervisar sus servicios, aplicaciones y productos de middleware.

Integrar

Red Hat JBoss Fuse
Bus de servicios empresariales (ESB) de tamaño reducido y flexible.
Red Hat JBoss A-MQ
Plataforma de mensajería de alto rendimiento y tamaño reducido.
Red Hat JBoss Data Virtualization
Plataforma de integración que unifica datos de fuentes dispares en una única fuente y expone los datos como servicio reutilizable.
Red Hat JBoss Fuse Service Works
Cartera de productos SOA que le permite crear, implementar, integrar y presentar aplicaciones y servicios.

Automatizar

Red Hat JBoss BRMS
Sólido sistema de gestión de reglas empresariales (BRMS) que proporciona a los responsables de la toma de decisiones de la empresa la habilidad para crear y cambiar las reglas empresariales de forma rápida.
Red Hat JBoss BPM Suite
Plataforma de gestión de decisiones y procesos empresariales que combina la gestión de reglas empresariales, la gestión de procesos empresariales (BPM) y el procesamiento de eventos complejos.

Relacionado

JBoss Community o JBoss Enterprise
Los desarrolladores interesados en Red Hat pueden elegir entre los proyectos de JBoss® Developer o Red Hat JBoss Middleware.
Migración a Red Hat JBoss Enterprise Middleware
Las empresas progresistas están cambiando los altos costos y las restricciones de los proveedores por el código abierto de Red Hat JBoss Middleware.

Virtualización

Red Hat Enterprise Virtualization
Gestión de virtualización empresarial completa para servidores y escritorios en la misma infraestructura.

Cloud computing

Red Hat Cloud Infrastructure
Oferta de suscripción única que le permite crear y gestionar una nube de infraestructura como servicio (laaS) abierta y privada, y acceder así a una infraestructura de tipo nube pública y con un alto grado de escalabilidad basada en OpenStack®.
Red Hat CloudForms
Plataforma de gestión de nube laaS integral que mejora sus infraestructuras de nube y virtuales con una planificación de capacidades avanzada y características de gestión de recursos.
Red Hat Enterprise Linux OpenStack Platform
Plataforma segura y escalable para la creación de nubes privadas y públicas.
OpenShift de Red Hat
Plataforma como servicio (PaaS) híbrida y abierta que permite que los desarrolladores creen, alojen, adapten y ofrezcan aplicaciones en la nube de forma rápida.
Red Hat Certificate System
Sistema de software empresarial que brinda un marco escalable y seguro para establecer y mantener identidades de confianza y conservar la privacidad de las comunicaciones.
Red Hat Directory Server
Registro basado en la red independiente del sistema operativo que permite a los administradores almacenar de manera central la identidad del usuario e información de la aplicación.

Relacionado

Nube privada
Cree hoy una nube privada con nuestra galardonada plataforma de gestión de la nube, la plataforma opcional de almacenamiento basado en software y la tecnología de virtualización de alto rendimiento y bajo costo.
Red Hat Cloud Access
Lleve con usted la suscripción Red Hat a la nube.

Almacenamiento

Red Hat Gluster Storage
Combina el fiable software de Red Hat con hardware básico x86, lo que elimina la necesidad de tener que acudir a costosos sistemas de almacenamiento patentados.
Red Hat Ceph Storage
Proporciona una plataforma sólida y altamente escalable para el almacenamiento de objetos y bloques para empresas que implementen nubes públicas o privadas. Votada como la solución n.º 1 de almacenamiento por los usuarios de OpenStack®.

Red Hat Mobile Application Platform
La movilización de procesos y aplicaciones empresariales pueden ayudarlo a aumentar su eficacia, a comprometerse con los clientes y a bajar costos.
