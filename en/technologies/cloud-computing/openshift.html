What is PaaS?

PaaS is a cloud application platform that automates the hosting, configuration, deployment, and administration of application stacks in an elastic cloud environment. It gives application developers self-service access so they can easily deploy applications on demand.

A platform that's prepared for whatever comes your way

Easily scale applications to handle increased traffic and demand. With OpenShift, you can deploy an application with scaling and high availability enabled by default.

Read the OpenShift Enterprise datasheet

Avoid lock-in with application portability

We built OpenShift with standards in mind. Any application written on OpenShift can easily be moved to another environment that supports the same languages, preventing vendor dependency and freeing you from lock-in.

Exploit the benefits of open source

Built on fully open source technologies, you control your particular implementation. You aren't restricted to the technology and business roadmap of a specific vendor—so you have visibility into the technology on which you're basing your business. And open source offers another benefit: the ability to collaborate with other communities and companies to drive innovation in areas that are important to you.

Build a stronger cloud on rock-solid technologies

OpenShift is built on proven, reliable Red Hat technologies, including Red Hat Enterprise Linux®—trusted by thousands of organizations on millions of systems around the world for mission-critical workloads—and Red Hat JBoss® Middleware.

PaaS models

The OpenShift platform is available in 2 different consumption models from Red Hat

Public PaaS

OpenShift Online is Red Hat’s hosted public PaaS that offers an application development, build, deployment, and hosting solution in the cloud.

  • Lets applications scale automatically in an elastic cloud environment
  • Faster time to market
  • Free sign-up: Available for developers to check out all the great features and capabilities, and start coding and running applications on OpenShift Online today
Learn more about OpenShift Online

Private PaaS

OpenShift Enterprise takes the same open source PaaS platform that powers the OpenShift Online hosted service and packages it for customers who want an on-premise or private cloud deployment.

  • A private PaaS that you control and manage within the constraints of your enterprise standards
  • Helps you increase developer productivity, streamline application service delivery, and achieve an elastic, on-demand cloud platform
